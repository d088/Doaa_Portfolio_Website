ITI Examination Management System

Full-cycle automation from registration to analytics using SQL Server, SSRS, and Power BI

Project Overview

This was our final graduation project at ITI â€” a complete Examination Management System built to automate and manage the entire academic examination lifecycle. Starting from student registration to exam scheduling, grading automation, and performance analytics, the system integrated SQL Server, .NET technologies, and BI tools into one seamless solution for academic institutions.

Features & Functionality

  • Student Registration: Secure onboarding with validation and duplicate checks.
  • Exam Scheduling: Automated conflict-free scheduling based on course and instructor availability.
  • Grading Automation: Instant score calculation for objective exams and result publishing.
  • Performance Analytics: Power BI dashboards for insights on student progress, pass rates, and instructor performance.

Data Architecture

Designed an optimized relational schema and ERD covering students, instructors, courses, exams, and results. Applied:

  • Normalization and referential integrity using PK/FK constraints
  • Stored Procedures for exam calculations and scheduling validations
  • Triggers for data consistency and audit trails
  • Indexes for high-performance querying

Reporting & BI

- Built 20+ Power BI dashboards with KPIs like top performers, pass/fail trends, and department-wise insights. - Designed SSRS reports for printable grade sheets, instructor reports, and exam summaries. - Used Power Query & DAX for advanced transformations and calculated metrics.
