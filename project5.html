Car Price Prediction

Predicting used car prices with Machine Learning & Streamlit

Project Overview

Buying a used car can feel like a gamble —The used car market is dynamic, with prices influenced by brand, mileage, fuel type, year of manufacture, and region. This project uses machine learning to analyze historical vehicle sales data and predict accurate price estimates — helping both buyers and dealers make data-driven decisions.

Dataset

We used the Vehicle Sales Data dataset from Kaggle. It contains 10,000+ vehicle listings with attributes like brand, year, transmission, fuel type, mileage, and price.

Methodology

The workflow combines Exploratory Data Analysis (EDA), Feature Engineering, and Random Forest Regression modeling:

Data Cleaning
Handle missing values & anomalies

EDA
Outlier detection, correlation heatmaps

Feature Engineering
Encoding, scaling, deriving new features

Modeling
Random Forest Regressor, hyperparameter tuning

Streamlit Dashboard
Interactive price prediction

Key Insights

Luxury Brands: BMW, Audi
Highest average resale values
Depreciation: Year & mileage
Strongest price influencers
Fuel Type: Diesel > Petrol
Diesel cars hold value longer
Model Performance: R² = 0.87
Accurate predictions with minimal error
