Car Price Prediction using ML

Data cleaning, EDA, and predictive modeling for vehicle prices using Python and Streamlit

Project Overview

This project involved analyzing and predicting vehicle prices through exploratory data analysis (EDA) and machine learning. The goal was to identify key factors influencing car prices and develop a predictive model using Python and Streamlit.

Machine Learning Model

A Random Forest Regressor was trained on cleaned and engineered data. Key features like mileage, engine size, car age, and fuel type were used. The model was evaluated with RÂ² and MSE and deployed to Streamlit for real-time predictions.

Streamlit Dashboard Features

Key Insights


Luxury Brands:
BMW, Audi retain highest resale value

Engine Size:
Larger engines drive higher prices

Transmission:
Manual cars depreciate slower in some regions

Fuel Type:
Diesel vehicles show higher resale than petrol

Age Factor:
Price decreases with age but not linearly
