Project Overview

This project focused on exploratory data analysis (EDA) and predictive modeling for vehicle prices. The goal is to analyze various factors affecting vehicle prices and build a model to predict prices based on these factors.

About The dataset:

  • Source: Kaggle – Vehicle Sales Data
  • Total Rows: ~10,000
  • Challenges: Missing values, inconsistent formatting, incorrect types (object instead of numeric) to estimating vehicle prices based on various attributes.
    • 🔗 View the Dataset

The interactive dashboard was built using Streamlit and Python to analyze and predict vehicle prices based on features such as brand, year, mileage, fuel type, and more. It offers visual EDA, feature exploration, and real-time prediction tools.

📊 Exploratory Data Analysis (EDA)

  • 🚘 Analyzed price distribution to understand value concentration across vehicle types.
  • 📅 Identified depreciation trends based on year of production.
  • 🛠 Correlation heatmap revealed strongest features affecting price were: mileage, engine size, and brand.
  • 🧹 Performed feature engineering such as converting categorical data to numerical form.

🧠 Machine Learning Model

A Random Forest Regressor model was trained to predict used car prices. After cleaning and preprocessing the data, relevant features such as mileage, engine size, car age, and fuel type were selected. The model was evaluated using R² and MSE, and showed strong performance in predicting prices accurately. The final model was deployed to Streamlit and responds in real-time to user inputs on the dashboard.

📈 Streamlit Dashboard Features

  • 📌 Sidebar filters for year, transmission type, and engine size.
  • 📉 Interactive plots showing price trends and outliers.
  • ⚙️ Regression model with prediction output displayed on input submission.

💡 Key Insights

  • Luxury brands and larger engines tend to have significantly higher resale prices.
  • Manual transmission vehicles slightly depreciate slower in some regions.
  • Vehicle age is inversely proportional to price but not always linearly.
  • Diesel cars generally have a higher price range than petrol or CNG models.
  • Engine size and mileage are among the most influential factors affecting price.
  • Luxury brands like BMW and Audi show significantly higher average resale values.
