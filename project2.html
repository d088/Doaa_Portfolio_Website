AdventureWorks2022 Analysis

From SQL Server restoration to actionable Power BI insights

How the Project Started

The AdventureWorks2022 dataset from Microsoft is a goldmine of realistic business data: customers, sales, products, and geography. To begin, I restored the database into SQL Server — a crucial first step to ensure access to the raw tables and relationships.

Data Restoration & Preparation

After restoring the ".bak" file into SQL Server, I explored the schema: multiple fact and dimension tables with relationships ideal for star schema modeling. Using SQL queries, I extracted clean subsets of data — such as total sales by geography, product performance, and customer demographics — ready for Power BI import.

Power Query & DAX in Action

Once in Power BI, I leaned on Power Query to merge tables, remove duplicates, and standardize data types. With DAX, I crafted custom measures — like profit margins, top product rankings, and year-over-year comparisons — turning raw numbers into actionable metrics.
