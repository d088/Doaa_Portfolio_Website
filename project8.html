AdventureWorks2022 Sales Analysis – Excel Dashboard

Turning Microsoft AdventureWorks data into actionable Excel insights

View Interactive Excel Dashboard on OneDrive

Project Overview

This project uses the AdventureWorks2022 dataset from Microsoft — a fictional retail company widely used for training and demonstrations. The goal: build a dynamic Excel dashboard to analyze sales performance, customer trends, and product categories with minimal manual effort.

Dataset

AdventureWorks2022 includes fact tables for sales and inventory and dimension tables for customers, products, and geography. Ideal for creating dashboards that mimic real business intelligence solutions in Excel.

Approach

I combined Power Query for data cleaning, PivotTables & PivotCharts for visualization, and Slicers & Timelines for interactivity — delivering an experience similar to Power BI, entirely inside Excel.
