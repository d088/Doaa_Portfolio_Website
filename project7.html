Netflix Movies & TV Shows Analysis

Uncovering global content trends with Power BI

Project Overview

Netflix has revolutionized entertainment by offering a vast library of movies and TV shows worldwide. This project analyzes the Netflix Titles Dataset to uncover content distribution, release trends, and genre popularity using Power BI.

Dataset

The dataset includes TV shows and movies with details like title, genre, release year, rating, duration, and country of origin. Perfect for exploring what content Netflix prioritizes and how it evolves over time.

Approach

I used Power Query to clean and structure the dataset, followed by DAX measures for advanced KPIs (e.g., content count by type, genre distribution, release frequency). Finally, I built an interactive Power BI dashboard to explore content patterns.
